App Overview



This Flutter application is a simple Task Manager that allows you to:

- Add new tasks  

- View all tasks in a list  

- Mark tasks as completed  

- View task details  

- Delete tasks with confirmation  





App Navigation Flow



1. Splash Screen - Welcome page with app logo.  

2. Home Screen - Displays list of tasks with Add button.  

3. Add Task Screen - Form to create a new task (title, description, priority).  

4. Task Detail Screen - Shows full task details with options to delete or mark as complete.







Features



Feature -- Description 



Add Task -- Create a new task with title, description, and priority 

Complete Task -- Mark any task as completed 

Delete Task -- Remove a task with confirmation dialog 

Navigation -- Implemented with `Navigator.push()` and `pushReplacement()` 

State -- Managed entirely with `setState()` in-memory 

UI Design -- Built with `Scaffold`, `AppBar`, `ListView`, and `Card` widgets 

Validation -- Task title is required before saving 

Feedback -- Uses `SnackBar` for add/delete messages
